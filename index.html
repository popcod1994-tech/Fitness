How the engine works
Live, in-session: log set 1's actual reps. If you beat the target by 2+, set 2's suggested weight nudges up slightly. If you miss by 2+, set 2 nudges down. Small, safe adjustments mid-workout.

Next session: after you save the workout, the average performance across all logged sets for each lift determines next time's prescription — up if you averaged at or above target, hold if close, deload 10% after two sessions of missing by 2+.

Timed holds (planks etc.): if you hit or beat the target time, next goal = your time + a small buffer (bigger buffer for short holds). If you fall short, the target repeats.
